# [Cell Shooter] — Game Concept

## Elevator Pitch

ผู้เล่นจะได้รับบทเป็นเม็ดเลือดขาวที่มีภารกิจกอบกู้ร่างกายที่ถูกโรคมะเร็งกัดกิน

## Genre & Platform

- **Genre:** 2D Action-Adventure
- **Platform:** PC (Windows)
- **Engine:** MonoGame (C#)
- **Target Audience:** เจาะจงไปที่ผู้เล่นที่ชื่นชอบเล่นเกม Soulknight และต้องการเสพเนื้อเรื่องนิดหน่อย

# [Sci-fi Upgrade] — Core Loop & Gameplay

## Core Loop

```mermaid
flowchart LR
    A[เริ่มต้น] --> B[ฆ่าศัตรู]
    B --> C[ได้รับชิ้นส่วน]
    C --> D[อับเกรดอาวุธ]
    D --> B
```

## Core Mechanics

1. ผู้เล่นจะต้องฆ่าศัตรูเพื่อรวบรวมชิ้นส่วน
2. ผู้เล่นสามารถนำชิ้นส่วนไปอัปเกรดความสามารถของอาวุธได้
